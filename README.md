# Implementaci-n-de-una-pila-en-C-aplicada-a-la-validacion-de-expresiones
Proyecto académico desarrollado en C++ que implementa una pila utilizando listas enlazadas para validar expresiones matemáticas simples.  El programa verifica el correcto balance de paréntesis, llaves y corchetes, así como la restricción de números de un solo dígito.
